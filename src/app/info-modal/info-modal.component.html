How is the Score Calculated?

The Machine Learning model implemented will return a prediction of:
  • 1: The model predicts that data can be collected from the source.
  • 0: The model predicts that data cannot be collected from the source.
The model also provides a probability for each of the predictions above.
If the prediction is 1(data can be collected from the source), the score will be the value of the prediction probability.
If the prediction is 0( data cannot be collected from the source), the score will be the negative value of the prediction probability.
This allows us to sort the score in a meaningful manner.
